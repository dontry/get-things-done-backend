# Skeleton project for Swagger

## install Typescript

```sh
yarn add typescript
tsc --init
```

## Create TSLint config

```sh
./node_modules/.bin/tslint --init
```

## Swagger API

```sh
yarn edit
```

```sh
yarn swagger-test
```

[Authorization Bearer](https://swagger.io/docs/specification/authentication/bearer-authentication/)
